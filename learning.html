INTRODUÇÃO AO HTML

HTLM é uma linguagem de marcação utilizada na construção de páginas web. Desenvolver websites. Quando acessamos uma aplicação na web na verdade estamos visualizando uma página HTLM. É hoje uma linguagem base para todos as aplicações web que utilizamos.

HTML (Hypertext Markup Language) Linguagem de marcação de Hypertexto.

TAGS

Tags são delimitadas pelo sinais de maior, menor e definem o significado de um determinado elemento presente no documento.

Para definir um parágrafo no meu documento. O p entre o sinais de menor e maior são os identificadores. Tudo que estiver no interior dessas duas tags será identificado por quem lê o documento HTML, como um parágrafo.

DOCUMENTO HTML

Um documento HTML é composto por diversas tags que, em conjunto, definem a estrutura e o conteúdo da página. E como ela deve ser lida. Tanto por um ser humano, como por uma máquina, que vai interpretar um documento. Normalmente, um navegador da internet.

Código 1

!DOCTYPE html

DOCTYPE é uma instrução sobre qual a versão do HTML que vai ser renderizada, nesse caso !DOCTYPE html representa o HTML versão 5. Essa declaração deve ser a primeira linha do arquivo HTML.

html

É dentro dessa tag que todas as demais serão escritas. A única coisa que pode ficar fora dessa tag é a declaração DOCTYPE. Um documento HTML deve ser escrito e estruturado em duas grandes seções: head e body.

head

É aqui que adicionamos tags com informações importantes sobre o documento, tais como o seu título, descrição, codificação de caracteres, entre outras. Nesse exemplo, utilizamos as tags meta e title.
